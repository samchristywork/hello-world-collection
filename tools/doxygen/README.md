## Overview

Doxygen version of "Hello, World!". The source file uses Doxygen comment
syntax (`@file`, `@brief`, `@param`, `@return`) to document the program. Run
`doxygen Doxyfile` to generate HTML documentation in the `docs/` directory.

## Example

```
$ doxygen Doxyfile
$ open docs/html/index.html
```

## Dependencies

```
doxygen
```

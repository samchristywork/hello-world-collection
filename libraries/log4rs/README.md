## Overview

This is the log4rs version of "Hello, World!". log4rs is a highly configurable
logging framework for Rust modelled after Java's log4j. This example configures
logging from a YAML file (`src/log.yaml`) and emits one message at each log
level: trace, debug, info, warn, and error. Log output is written to
`log/my.log`.

It can be built and run with `cargo`.

## Running

```
cargo run
```

## Dependencies

```
cargo
rust
```

## Overview

This is the Glade version of "Hello, World!". It uses GtkBuilder to load a UI
layout defined in a `.glade` XML file, rather than constructing widgets in code.
When the button is clicked, "Hello, World!" is printed to stdout.

It can be built with `make`.

## Dependencies

```
gcc
glade
libgtk-3-dev
make
pkg-config
```

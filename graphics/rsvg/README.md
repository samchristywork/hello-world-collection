## Overview

rsvg version of "Hello, World!". librsvg is a library for rendering SVG files,
developed by the GNOME project. This example defines "Hello, World!" as text in
an SVG file and uses `rsvg-convert` to render it to a PNG image in the `build`
directory.

Build with `make`. The output will be in the `build` directory.

## Dependencies

```
librsvg2-bin
make
```

## Overview

This is the wxFormBuilder version of "Hello, World!". wxFormBuilder is a GUI
designer for wxWidgets that generates C++ code from a visual layout. The
generated code lives in `generated_code/` and is compiled alongside `hello.cpp`,
which subclasses the generated frame and connects event handlers.

It can be built with `make`.

## Dependencies

```
g++
libwxgtk3.0-gtk3-dev
make
wxformbuilder
```

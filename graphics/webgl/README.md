## Overview

This is the WebGL version of "Hello, World!". It renders a single large red
square at the center of a canvas using a vertex shader and a fragment shader
written in GLSL. The shaders are embedded directly in `index.html` as script
tags with custom MIME types.

There is no build step. Open `index.html` in a browser that supports WebGL.

## Dependencies

```
A WebGL-capable browser (e.g. Firefox, Chrome)
```
